## Initialize Kafka Partitions

This project uses a helper script to alter the number of partitions for a Kafka topic.

### Usage

- **Windows PowerShell**  
  bash scripts/alter_partitions.sh raw_links 2
- **Git Bash**
  ./scripts/alter_partitions.sh raw_links 2
